Check OmniStudio package version#

Salesforce releases updates to the OmniStudio Managed Package during every release cycle i.e. Spring, Summer, and Winter. Updates to the Omnistudio Managed Package should happen automatically in your production org.

In the event that you have a trail org that has an outdated version, or your production org hasn’t updated automatically, you can manually update the managed package.

  1. Check latest version by visiting the OmniStudio Release page

  2. Check which version is in your org:

    1. Navigate to Setup

    2. Search Installed Packages

    3. Search for Package Name Omnistudio

    4. Check version number

    1. For outdated versions, follow the instructions in the OmniStudio Release page to install the latest version.

    2. After the updates/installation has completed, verify that the Installed Package version is correct.
